# httpie-curl-examples

Working demo on how to use httpie and curl terminal commands (HEAD, GET, PUT, POST, DELETE)

# Prerequisites

- [curl](https://curl.se/)
- [httpie](https://httpie.io/)
- [node.js v22](https://nodejs.org)

# Suggested

- [volta](https://volta.sh/)

# Install

```
npm install
```

# Run server

```
npm start
```
